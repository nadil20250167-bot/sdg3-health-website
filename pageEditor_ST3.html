Technical Description

HTML5 Structure

I structured feedback.html using two main semantic modules. First, a scrollable list containing health opportunities. Second, an interactive form grouped by <fieldset> and <legend> tags separating personal data from feedback content. On team.html, I structured 4 profile blocks with custom avatar parameters.

CSS Techniques & Layout

I designed the team page using pure CSS hover details. By default, card details are collapsed using max-height: 0; opacity: 0;. On hover or keyboard focus, these values expand to display the full description:

.member-card:hover .contribution-details,
.member-card:focus-within .contribution-details {
    max-height: 200px;
    opacity: 1;
}

JavaScript Functionality

I programmed the JS form validator which runs on submission. If inputs are missing or incorrect, it intercepts the submit event, highlights borders in red, and updates descriptive warning elements. I also coded a live text area character tracker:

function updateCharCount() {
    const left = 150 - comment.value.length;

Accessibility Considerations

I placed explicit <label> attributes on all form inputs and added tabindex="0" to the team cards. This allows visually impaired students utilizing screen readers to navigate and hear member contribution summaries easily.
